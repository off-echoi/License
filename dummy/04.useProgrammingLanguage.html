126. 데이터 타입(B)

  • 정수 타입 : 소수점 없는 숫자
  • 부동소수점 타입 : 실수, 소수점 있는 숫자
  • 문자 타입 : 한 문자, '' 사용
  • 문자열 타입 : 문자열 , "Hello" 사용
  • 불린 타입 : 조건의 참 거짓, 기본은 거짓
  • 배열 타입 : 데이터 집합, { , , }로 구분

127. 변수

  • 컴퓨터가 명령처리 도중 발생하는 값을 저장하기 위한 공간
  • 정수형, 실수형, 문자형, 포인터형

  1. 변수명 작성 규칙 / C언어 기준

    • 영문, 숫자, _(uder bar) 사용
    • 첫글자는 숫자가 올 수 없음
    • 글자수 제한 없음
    • 공백, *,+,-,/ 등 특수문자 사용 불가
    • 대,소문자 구분
    • 변수 선언 문자 끝에 반드시 세미콜론(;)
    • 예약어 사용 불가
      제어문
      [do, for, while], [case, default, else, if, switch], [break, continue, goto, return]
      자료형
      char, double, enum, float, int, long, short, signed, struct, typedef, union, unsigned, void
      기억 클래스
      auto, extern, register, static
      기타
      const, sizeof, volatile

  2. 기억 클래스

    • 기억영역을 결정하는 작업
    • 변수선언시 할당되는 기억영역에 따라 사용범위 제한
    종류 설명 기억영역 예약어 생존기간 사용 범위
    자동변수
    (Automatic Variable)    		 블록 내에 선언
    함수나 블록이 실행될때만 존재
    초기화하지 않으면 쓰레기값저장    		 메모리(스택) auto(생략가능) 일시적 지역적
    레지스터 변수
    (Register Variable)    		 자주사용 변수 저장 => 처리속도 향상
    블록스코프
    사용개수한정, 다 쓰면 자동으로 메모리에 할당
    메모리주소가 없기때문에 변수의 주로를 구하는 연산자 사용불가    		 레지스터 register
    정적변수
    (Static Variable)     		내부 변수 : 블록스코프
    외부 변수 : 전역
    값 소멸 안됨
    변수 선언과 동시에 초기화, 생략하면 0    		 메모리(데이터) static 영구적 내부 : 지역적
    외부 : 전역적
    외부변수 다른 변수를 참조하기위한 변수
    함수밖에서 선언
    종료뒤에도 소멸 안함, 초기화안하면 0
    다른 변수를 참조했을 경우 초기화 못함    		 extern 전역적
